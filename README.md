# GestureApp

A powerful Flutter application that allows users to automate tasks through custom screen gestures and codes.

## Features
- **Gesture Recognition**: Draw custom shapes to trigger actions.
- **Quick Codes**: Execute commands via simple text codes.
- **Floating Bubble**: Access gesture canvas from any screen with a floating trigger.
- **Customizable**: Adjust opacity, size, and themes for the gesture bubble.
- **Syncing**: Securely sync your gestures with a Laravel-powered backend.

## Tech Stack
- **Frontend**: Flutter (Mobile)
- **Backend**: Laravel (API & Admin Panel)
- **Database**: MySQL & Hive (Local Caching)

## Getting Started
1. Clone the repository.
2. Follow the setup instructions for both `gesture_app` (Flutter) and `gesture-admin` (Laravel).
3. Enjoy automating your mobile experience!
